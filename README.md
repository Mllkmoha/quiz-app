# Quiz App

Interactive React quiz application with timer, answer shuffling, and detailed results summary.

## Features

- ⏱️ Dynamic timer that adjusts based on user interaction
- 🔀 Shuffled answer options
- ✅ Real-time feedback (correct/wrong/skipped)
- 📊 Summary screen with statistics
- 📱 Responsive design

## Components

- `Quiz.jsx` - Main quiz logic
- `Question.jsx` - Question display with timer
- `Answers.jsx` - Shuffled answer buttons
- `Summary.jsx` - Results summary screen
- `QuestionTimer.jsx` - Timer progress bar

## Installation

```bash
npm install
npm run dev
```

## Technologies

- React (Hooks: useState, useCallback, useRef, useEffect)
- Vite
- CSS3

## How to Use

1. Start the quiz
2. Answer questions before timer expires
3. See instant feedback
4. View final summary with statistics
